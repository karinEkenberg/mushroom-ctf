# Mushroom CTF - Metadata Challenge

A beginner-friendly Capture The Flag (CTF) challenge focused on steganography and metadata analysis.

## Objective
Investigate the provided file to find the hidden flag. The flag format is `CTF{...}`.

## Setup
Download `challenge.zip` to start. Always use the zipped version to prevent metadata stripping by operating systems or sharing platforms.

## Recommended Tools
- ExifTool
- Base64 decoding utilities
- Linux terminal or WSL

## Solution
A full step-by-step solution is available in `writeup.md`. Try to solve the challenge before looking at the answers.
